# VTHO Balance Ticker Chrome Extension

## Development

* Run script
```bash
  # build files to './dev'
  # start webpack development server
  $ npm run dev
```

## Build
```bash
  # build files to './build'
  $ npm run build
```


## Compress
```bash
# compress build folder to {manifest.name}.zip and crx
$ npm run build
$ npm run compress -- [options]
```
